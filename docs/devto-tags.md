# Recommended dev.to Tags

When publishing the launch post on dev.to, we recommend using the following tags to maximize reach and target the right audiences:

## Primary Tags
1. `#ai` (Very active community of developers, researchers, and hobbyists building with AI/LLMs)
2. `#python` (Since the reference server, FastMCP wrapper, and client SDK are all written in Python, this is highly relevant)
3. `#opensource` (The core protocol specification, server, and client SDK are completely MIT licensed and open to contribution)
4. `#webdev` (As AMP is an HTTP-native REST protocol, it appeals to backend and full-stack developers designing multi-agent communication networks)

## Alternative / Secondary Tags (If platform limits differ)
- `#architecture` (Useful to discuss the cognitive memory architectures and lifecycle models)
- `#programming` (General programming audience)
- `#showdev` (Great for sharing new libraries, tools, and protocols)
